# Portfolio — Alan Fernández Diosdado

Portfolio personal donde muestro mis proyectos, stack tecnológico y forma de trabajar como desarrollador Fullstack Junior.

---

## Secciones

- **Hero** — Presentación e introducción
- **Sobre mí** — Formación, experiencia y enfoque
- **Proyectos** — Proyectos destacados con código y demo
- **Skills** — Stack tecnológico por categorías (Frontend, Backend, Base de datos, DevOps)
- **Contacto** — Enlace directo para contactar por email

## Características

- 🌐 **Multilenguaje** — Español e inglés, con selector persistente (localStorage)
- 🎨 **Tema oscuro** — Diseño minimalista con acento violeta
- ✨ **Animaciones** — Scroll reveals, contadores animados, efecto scramble en títulos
- ♿ **Accesible** — Etiquetas ARIA y `lang` sincronizados con el idioma activo

## Stack

- **Framework:** Next.js + React + TypeScript
- **Estilos:** Tailwind CSS
- **Animaciones:** GSAP
- **Deploy:** Vercel

## Proyectos destacados

- 🎬 **Cinesfera** — Red social cinematográfica con búsqueda de películas y series, reseñas, listas personalizadas y recomendaciones por IA
- 🤖 **RAG para Cinesfera** — Sistema de memoria persistente que mejora las recomendaciones con el tiempo mediante RAG
